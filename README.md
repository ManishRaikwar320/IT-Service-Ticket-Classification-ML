# 🎫 IT Service Ticket Classification & Prioritization

## 🚀 Project Overview
This project builds a Machine Learning system to automatically classify IT service support tickets and assign priority levels.

In real-world organizations, support teams receive a large number of tickets daily. This system helps automate ticket categorization and prioritization, enabling faster resolution and improved efficiency.

---

## 🎯 Objective
- Automatically classify support tickets into categories  
- Assign priority levels (High / Medium / Low)  
- Reduce manual effort in ticket management  
- Improve response time and customer satisfaction  

---

## 🧠 Key Features
- ✔ Text Cleaning & Preprocessing  
- ✔ Feature Extraction using TF-IDF  
- ✔ Machine Learning-based Classification  
- ✔ Rule-based Priority Assignment  
- ✔ Model Evaluation  

---

## 📊 Workflow
1. Data Loading  
2. Text Cleaning (lowercase, stopword removal, punctuation removal)  
3. Feature Extraction (TF-IDF)  
4. Model Training  
5. Prediction  
6. Priority Assignment  
7. Model Evaluation  

---

## 🤖 Model Used
- Logistic Regression (Primary Model)  
*(Decision Tree / RandomForest)*
